¿Que es un Pendrive Booteable?

Un USB booteable es una memoria en formato USB que sirve para llevar a cabo el proceso de 'boot' o lo que es lo mismo, arrancar el ordenador desde el USB.

Windows

Microsoft Windows es una gama de sistemas operativos gráficos que domina el mercado de la informática personal. Conocido por su interfaz fácil de usar, el sistema operativo Windows ha transformado la forma en que los usuarios interactúan con sus dispositivos, especialmente en un ordenador con Windows. Con una gran interfaz gráfica de usuario (GUI) y capacidades multitarea, Windows se ha consolidado como el sistema operativo de referencia tanto para usuarios ocasionales como profesionales.
